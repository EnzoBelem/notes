
[[Computação|Início]]

Docker é uma plataforma de código aberto que simplifica a criação, o gerenciamento e a execução de aplicações em **containers**. Containers são ambientes isolados que agrupam o código da aplicação com todas as suas dependências, garantindo consistência entre diferentes ambientes, sejam eles de desenvolvimento, teste ou produção.
